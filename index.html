Розв’язання квадратнго рівняння типу:

ax2 + bx + c = 0

Для того, щоб рівняння мало два корені, можна ввести наступні значення змінних: а = -6, b = 1, c = 1

Для того, щоб рівняння мало один корінь, можна ввести наступні значення змінних: а = 1, b = 2, c = 1

Для того, щоб рівняння не мало коренів, можна ввести наступні значення змінних: а = 5, b = 1, c = 3
